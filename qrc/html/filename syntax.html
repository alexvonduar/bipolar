Filename Format

The output filename may be specified by a format string. The format string may use any combination of the following expressions, which will be replaced according to the following table:

ExpressionOutput
$baseName the training session's base name (e.g. v2-users-123-training-sessions-456)
$date the training session's local date, in ISO 8601 basic format (e.g. 20140816)
$dateExt the training session's local date, in ISO 8601 extended format (e.g. 2014-08-16)
$dateUTC the training session's UTC date, in ISO 8601 basic format (e.g. 20140816)
$dateExtUTC the training session's UTC date, in ISO 8601 extended format (e.g. 2014-08-16)
$time the training session's local time, in ISO 8601 basic format (e.g. 06:30:00)
$timeExt the training session's local time, in ISO 8601 extended format (e.g. 06:30:00). Warning: not supported on most Windows filesystems.
$timeUTC the training session's UTC time, in ISO 8601 basic format (e.g. 063000)
$timeExtUTC the training session's UTC time, in ISO 8601 extended format (e.g. 06:30:00). Warning: not supported on most Windows filesystems.
$userId the training session's Polar user ID (e.g. 12345678)
$username the name of the current user (e.g. Paul)
$sessionId the training session's ID (e.g. 23456789)
$sessionName the training session's name (e.g. "Running", "Cycling", "Multisport", etc.)

Text not matching any of the above expressions is used verbatim.

If no format string is provided, the default is to use the original training session's base name (i.e. "$baseName"). A commonly preferred variant is: "$date $time $sessionName"
